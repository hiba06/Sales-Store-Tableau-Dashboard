# Project Title

Sales Store Data Analysis using Tableau

---

## Dataset Used

- <a href="https://github.com/hiba06/Sales-Store-Tableau-Dashboard/blob/main/Orders.xlsx">Dataset</a>

---

## Project Objective

1. To analyze sales store data using Tableau.
2. To understand sales performance across different countries.
3. To identify top-performing salespersons.
4. To track shipment and box delivery performance.
5. To support data-driven business decision making.

---

## Business Questions

1. What is the total sales amount generated by each country?
2. Which country has the highest sales revenue?
3. What is the total number of shipments and boxes shipped?
4. Which salesperson achieves the highest sales?
5. How does sales performance vary across regions?

---

## Key Performance Indicators (KPIs)

1. Total Sales Amount.
2. Total Boxes Shipped.
3. Total Shipment Count.
4. Sales Amount by Country.
5. Sales Amount by Salesperson.

---

## Process

1. Collected the sales store dataset.
2. Cleaned and structured the data for analysis.
3. Imported the dataset into Tableau.
4. Created calculated fields for sales, shipments, and boxes shipped.
5. Designed KPI summary section.
6. Built bar charts for sales by country.
7. Built interactive charts for sales by salesperson.
8. Combined all sheets into a single interactive dashboard.
9. Published the dashboard on Tableau Public.
